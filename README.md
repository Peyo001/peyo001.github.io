Ce projet est un site web de portfolio personnel réalisé dans le cadre de mes études en BUT Informatique. Il me permet de présenter mon parcours, mes compétences, mon CV et mes projets réalisés, dans une interface moderne et responsive.

Le site est développé en HTML, CSS, JavaScript et utilise le framework Bootstrap 5 pour la mise en page. Il inclut également un mode sombre, un affichage du CV dans une fenêtre modale, un défilement fluide entre les sections, et plusieurs cartes de présentation (projets, compétences, langues, centres d’intérêt, etc.).

Fonctionnalités

Visualisation et téléchargement du CV

Mode sombre (avec sauvegarde via localStorage)

Défilement fluide vers les sections du site

Présentation de projets

Affichage des compétences, formations, langues, centres d’intérêt

Design responsive (adapté aux écrans mobiles)

Structure du projet

Le projet est composé des fichiers suivants :

index.html : page principale du site

style.css : feuille de styles personnalisée

cv.pdf : mon CV au format PDF

node_modules/ : dépendances (notamment Bootstrap)

README.md : ce fichier

Technologies utilisées

HTML5

CSS3

Bootstrap 5

JavaScript 

Font Awesome (icônes)

Comment l’exécuter en local

Cloner le dépôt :
git clone https://github.com/Peyo001/Portofolio.git

cd Portofolio

Installer les dépendances si nécessaire :
npm install

Ouvrir le fichier index.html dans un navigateur :
soit en double-cliquant sur le fichier, soit en utilisant une extension comme "Live Server" sur VS Code.
