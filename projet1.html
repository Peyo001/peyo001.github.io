Projet : Site Portfolio

Site portfolio réactif (français) avec thème noir/or, icônes Font Awesome, animations et mise en page moderne — conçu pour présenter mes compétences, CV et projets.

Description

Ce projet présente la version actuelle de mon portfolio, développée avec HTML, CSS, JavaScript et Bootstrap. Points clés :

  • Thème noir et or cohérent sur toutes les pages
  • Icônes Font Awesome pour les boutons, compétences et liens
  • Badges pour les compétences et cercles animés pour représenter les niveaux
  • Sections principales : Profil, CV, Compétences, Projets et Contact
  • Pages de projet en français et boutons de navigation vers chaque page

Technologies utilisées

  • HTML / CSS (variables CSS, responsive)
  • JavaScript (animations, IntersectionObserver, smooth-scroll)
  • Bootstrap 5 (layout, composants)
  • Font Awesome (icônes)

Difficultés rencontrées

  • Harmoniser le thème noir/or et assurer un contraste accessible
  • Garder les badges lisibles sur mobile
  • Synchroniser les textes des pages projets en français
