Projet 3 : Analyse de données web avec Python

Ce projet analyse les données de trafic d'un site web à l'aide de Python, SQL et d'outils de visualisation comme Matplotlib.

Description

Le programme se connecte à une base de données via ODBC et exécute plusieurs requêtes SQL en fonction des paramètres fournis. Il permet de :

  • Obtenir la page la plus visitée pour un pays et une année donnés
  • Afficher les pays et pages les plus actifs par saison
  • Afficher les visites uniques par activité (randonnée, escalade, ski, etc.)
Les résultats sont affichés en console et visualisés à l'aide de graphiques Matplotlib.

Technologies utilisées

  • Python
  • SQL
  • ODBC
  • Matplotlib

Difficultés rencontrées

  • Établir une connexion ODBC stable via DSN
  • Construire dynamiquement des requêtes SQL en fonction des choix
  • Créer des visualisations claires avec Matplotlib
  • Gérer les saisies utilisateur invalides (saison erronée, options incorrectes, etc.)
