Project 4: CD Player with Qt and C++

A desktop application developed using the Qt framework that simulates a functional CD player with basic media controls.

Description

This project demonstrates how to create a simple graphical CD player using Qt in C++. The application allows users to:

  • Load and play audio tracks (e.g. MP3, WAV)
  • Pause, resume, stop playback
  • Navigate between tracks using Next/Previous buttons
  • Display current track title and playback progress
It was built as part of my coursework to deepen my understanding of GUI development, event handling, and multimedia integration in C++ using Qt.

Technologies Used

C++ Qt Framework Multimedia (QMediaPlayer) GUI Design

Features

  • Responsive user interface with Qt Widgets
  • File dialog to select audio files
  • Custom-designed control buttons (Play, Pause, Stop)
  • Volume and progress sliders
